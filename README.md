# SmartTasker — 手机超级管家

> 一句话，搞定一切。你的手机自动化超级助手。

> ⚠️ **第一原则**：本项目以 [SmallvL/SmartTasker](https://github.com/SmallvL/SmartTasker) 为唯一权威上游。任何决策与改动须遵守 [PRODUCT_DIRECTION.md](./PRODUCT_DIRECTION.md)。本地旧项目 `D:\1_AIagnet\SMARTTASK\android\` 已 DEPRECATED。

SmartTasker 是一个 Android 端的超级管家，类似超级 Siri、超级小爱同学。用自然语言描述你想要完成的事情，SmartTasker 就能自动帮你在手机上执行。

**核心理念**：像和真人管家说话一样操控手机。

```
"每天早上9点打开企业微信打卡"  →  自动创建定时任务，准时执行
"收到老板微信后打开查看"      →  监听通知，自动响应
"打开设置检查系统更新"        →  自动导航，一键完成
```

## ✨ 核心能力

- **🗣️ 自然语言交互** — 用日常语言描述任务，AI 自动解析为可执行的操作序列
- **⏰ 智能定时** — 支持精确闹钟调度，开机自动恢复，永不遗漏
- **🔔 通知触发** — 监听手机通知，满足条件自动执行
- **🎬 路线录制** — 录制一次操作，永久复用，支持编辑和优化
- **🛡️ 安全策略** — 风险评估、确认机制、操作沙箱
- **📊 数据洞察** — 运行统计、成功率追踪、失败诊断

## 🏗️ 技术架构

- **Kotlin + Jetpack Compose** — 现代 Android UI
- **ADB / Shell 执行引擎** — 三种模式：ADB TLS / Root / SH
- **Room Database** — 本地数据持久化
- **Hilt DI** — 依赖注入
- **LLM 集成** — 支持 GPT-4o-mini 等模型进行任务解析

## 📱 使用场景

| 场景 | 示例 |
|------|------|
| 定时打卡 | "每天早上9点打开企业微信打卡" |
| 消息响应 | "收到微信消息后自动打开查看" |
| 系统维护 | "每周清理一次手机存储" |
| 应用操作 | "打开淘宝领取每日金币" |
| 批量任务 | "依次打开5个APP签到" |

## 🚀 快速开始

1. 安装 APK
2. 配置 ADB 连接（无线调试或 Root）
3. 开始对话，描述你想自动完成的事
