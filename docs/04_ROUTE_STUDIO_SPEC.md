# 04. Route Studio 路线编辑器规格

> 项目：SmartTask AI / AI 安卓自动化任务产品  
> 版本：v0.2  
> 日期：2026-05-23  
> 底层参考：AutoLXB 二次开发

## 1. 模块定位

Route Studio 是产品的核心差异化模块。

它不是 AI 自动纠错黑盒，而是：

> **AI 辅助的可视化安卓自动化路线编辑器。**

核心原则：

```text
人工可控编辑为主
AI 建议提效为辅
所有修改可撤销、可测试、可回滚
```

---

## 2. 用户目标

用户进入 Route Studio 时通常有三类目标：

| 目标 | 示例 |
|---|---|
| 看懂路线 | “这个任务到底做了哪些操作？” |
| 修正路线 | “第 3 步点错了，我要改成点另一个入口。” |
| 提高稳定性 | “把坐标点击改成文本定位。” |

## 4. 步骤时间线

### 4.1 步骤状态

| 状态 | 图标建议 | 含义 |
|---|---|---|
| 成功 | ✅ | 最近执行成功 |
| 警告 | ⚠️ | 稳定性低或风险中等 |
| 失败 | ❌ | 最近执行失败 |
| 禁用 | ⏸ | 用户暂时跳过 |
| 锁定 | 🔒 | 用户锁定，不允许 AI 覆盖 |

### 4.2 步骤操作

MVP 必须支持：

| 操作 | 说明 |
|---|---|
| 查看详情 | 查看截图、属性、定位方式 |
| 删除 | 从路线中移除 |
| 禁用 | 不执行但保留 |
| 修改摘要 | 修改步骤说明 |
| 修改点击目标 | 重新选择控件/坐标/文本 |
| 修改等待时间 | 调整 timeout / wait |
| 单步测试 | 验证当前步骤 |
| 保存草稿 | 保存编辑但不发布 |
| 发布版本 | 设置为任务执行路线 |

P1 支持：

- 拖拽排序。
- 复制步骤。
- 插入步骤。
- 锁定步骤。
- 分组步骤。
- 查看版本 diff。

## 5. 截图交互区

### 5.1 展示内容

| 元素 | 说明 |
|---|---|
| 原始截图 | 当时执行该步骤的屏幕 |
| 点击点 | 原路线点击坐标 |
| 命中控件框 | Accessibility/XML 匹配区域 |
| 候选控件框 | 可替换目标 |
| OCR 文本框 | 文本定位候选 |
| 风险遮罩 | 高风险区域可特殊提示 |

### 5.2 用户操作

| 操作 | 结果 |
|---|---|
| 点选控件框 | locator 改为 resource-id/content-desc/text |
| 点选 OCR 文本 | locator 改为 text |
| 拖动点击点 | 修改 coordinate fallback |
| 长按截图 | 显示更多定位候选 |
| 双指缩放 | 查看细节 |

## 6. 步骤属性编辑

### 6.1 点击步骤

字段：

```text
动作类型：点击
步骤摘要
主定位策略
备用定位策略
重试次数
重试间隔
失败兜底
风险等级
是否需要确认
是否用户锁定
```

定位策略优先级建议：

```text
resource-id / content-desc / text
 -> visual description
 -> coordinate fallback
```

### 6.2 等待步骤

支持：

- 固定等待。
- 等待文本出现。
- 等待控件出现。
- 等待页面稳定。
- 最大超时。

### 6.3 输入步骤

支持：

- 固定文本。
- 用户运行时输入。
- 通知内容变量。
- AI 生成文本。
- 变量引用。

涉及发送时，发送动作必须进入安全确认。

## 7. AI 辅助规则

AI 可以：

- 解释路线。
- 标注低稳定步骤。
- 标注高风险动作。
- 生成修复建议。
- 建议批量优化等待。
- 建议替换 locator。

AI 不可以：

- 静默修改用户路线。
- 覆盖用户锁定步骤。
- 自动发布新版本。
- 自动执行高风险动作。

AI 建议卡片必须有：

```text
建议内容
原因
影响
[应用建议] [忽略] [查看详情]
```

## 8. 单步测试

### 8.1 MVP 目标

### 8.2 测试方式

## 9. 版本管理

### 9.1 状态

| 状态 | 说明 |
|---|---|
| draft | 草稿，可编辑 |
| published | 当前执行版本 |
| archived | 历史版本 |
| candidate | AI 修复候选版本 |

### 9.2 版本来源

| 来源 | 说明 |
|---|---|
| ai_learned | 首次 AI 学习生成 |
| user_edit | 用户人工编辑 |
| ai_suggestion_applied | 用户应用 AI 建议 |
| fallback_repair_candidate | 路线失败后视觉兜底成功产生候选 |

### 9.3 版本操作

- 保存草稿。
- 发布草稿。
- 回滚旧版本。
- 从历史版本复制为草稿。
- 查看修改摘要。
